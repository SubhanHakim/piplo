# PIPLO

A minimalist 8-bit / NES-inspired landing page for the PIPLO project.

## Overview

PIPLO is a small 8-bit entity that never shut down. This landing page captures a nostalgic, consistent, and quiet atmosphere using a retro terminal aesthetic.

## Features

- **Retro Aesthetic**: Pure black background, pixel-perfect rendering.
- **Typography**: VT323 Google Font for terminal-style readability.
- **Responsive**: Minimal, functional layout that works on mobile.
- **Pixel Art**: Custom generated 8-bit sprite.

## How to Run Locally

This project is built with React and Vite.

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Navigate to the URL shown in the terminal (usually `http://localhost:5173`).

## Structure

The source code is located in `src/`.
- `src/App.tsx`: Main component containing the layout and content.
- `src/App.css`: All retro styling.
- `src/assets/piplo.png`: The pixel art sprite.
